# javachat-client_server_socket_project
Javachat is a small project between a group of friends where we experiment with Java socket servers hosted across a private network.

FEATURES:
- Members can all communicate with eachother using an intuitive GUI
- Concurrency allows for multiple members to communicate at the same time
- Direct Messages can be sent to individual members
- A server 'coordinator' is selected who will automatically manage user activity, IP's, Ports & usernames along with tracking message timestamps.
- Members can view other members private IP's and ports by clicking on their username.
- A list of users who are online is provided in the GUI.
